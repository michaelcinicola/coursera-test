Div is the most common Block-level element. Blocks render to begin on a new line by default.
Span is the most common Inline-level element. Spans render to begin on the same line by default.
Div 1 = new line
Div 2 = new line
Span 1 = same line (in-line) BUT the Div above it has a behavior that renders itself on its own line, pushing the span down a line.
Div 3 = new line (despite following a span) Span 2 = In-line BECAUSE this span is inside the div before the div is closed. So no pushing the span down a line this time.Hi
